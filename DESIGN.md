# Design System

## Direction

克制的桌面生产力工具。以白色内容面、浅灰表头、细分隔线和青色主操作构成。整体接近专业脚本编辑器，不使用营销页面结构。

## Typography

- 字体：系统无衬线字体栈。
- 应用标题：16px / 700。
- 表头：12px / 600。
- 正文和输入：13px / 400。
- 数字列使用等宽数字。

## Color

- 页面背景：中性浅灰。
- 内容表面：白色。
- 主文字：近黑。
- 次级文字：中灰。
- 分隔线：浅灰。
- 主操作与焦点：青色。
- A-ROLL：蓝色；B-ROLL：橙色；素材就绪：绿色。

## Layout

- 顶栏固定为 64px。
- 表格紧接顶栏，不设置宣传区。
- 表格区域独立滚动，表头固定在表格容器顶部。
- 使用 4px 基础间距系统。

## Components

- 按钮：8px 圆角，清晰 hover、active、focus-visible 状态。
- 自定义下拉：按钮触发，弹层通过 fixed portal 挂载到 body。
- 文本域：透明默认态，hover 出现浅灰底，focus 显示青色边框。
- 素材预览：8px 圆角，16:9 画面，空状态简短。

## Motion

仅用于下拉菜单出现和保存状态，时长 120–180ms。遵循 `prefers-reduced-motion`。
